**Описание проекта** 
---


Сервис для работы с базой данных интернет магазина.

## **Установка и запуск** 

- Склонировать репозиторий: 'git clone https://github.com/Kiri11Uroven/PcStoreApi'

- Открыть код в редакторе

- Запустить проект
